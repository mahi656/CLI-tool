# CLI Tool

A simple and lightweight Command-Line Interface (CLI) application built using Node.js.

## Installation

1. Make sure you have Node.js installed.
2. Run `npm install` in this folder.

## How to run commands

Use `npm start -- <command>` to run the tool.

### Examples

**Basic Commands:**
- `npm start -- greet Mahi` - Greets you!
- `npm start -- fileinfo data.txt` - Shows file size

**Math:**
- `npm start -- add 5 10` - Adds two numbers
- `npm start -- sub 20 5` - Subtracts numbers
- `npm start -- mul 4 5` - Multiplies numbers
- `npm start -- div 100 4` - Divides numbers

**APIs:**
- `npm start -- weather Tokyo` - Gets the current weather
- `npm start -- github torvalds` - Gets user GitHub info
- `npm start -- quote` - Gives you a random quote
- `npm start -- joke` - Tells a programming joke
